# Phase Essentials — Overview

Phase Essentials are 1-pager quick reference cards for each phase. Each card answers three questions:
1. **What to do** — Key activities in this phase
2. **Who** — Primary roles and accountability
3. **Evidence required** — What artefacts prove the phase is done

## Purpose

Phase Essentials are the first thing every phase agent reads before starting. They serve as a fast orientation to:
- Avoid reading all reference docs upfront
- Get the 80% of context needed to start in under 2 minutes
- Link to the right templates and skills quickly

## Available Phase Essentials

Phase Essentials cards are created by each phase agent at the start of their phase. They live at:
- `docs/phase-essentials/phase-1.md` — Discovery & Framing

## Format

Each card follows this structure:

```markdown
# Phase N — [Name] Essentials

## What to Do (5 bullets max)
- ...

## Who
| Role | Responsibility |
| ... | ... |

## Evidence Required for Gate [X]
| Artefact | Template | Evidence Level |
| ... | ... | ... |

## Key Skills
- skill-name — when to use

## Entry Criteria (1 line each)
- ...

## Exit Criteria (1 line each)
- ...
```

## Usage

Phase agents automatically reference the relevant essentials card. You can also read them directly when starting work on a phase.
