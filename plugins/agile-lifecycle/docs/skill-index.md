# Skill Index

All 16 skills in the agile-lifecycle plugin with trigger conditions.

| Skill | Directory | Triggers When |
|-------|-----------|--------------|
| phase-contract | `skills/phase-contract/` | Starting a new phase, validating entry/exit criteria, checking phase completeness |
| gate-checklist | `skills/gate-checklist/` | Executing a formal gate review (A-F), checking gate criteria, deciding PASS/FAIL/WAIVED |
| artefact-authoring | `skills/artefact-authoring/` | Creating or filling lifecycle artefacts, selecting templates, validating completeness |
| risk-management | `skills/risk-management/` | Adding/updating risks, assumptions, clarifications, dependencies |
| metrics-tracking | `skills/metrics-tracking/` | Tracking, calculating, or reporting lifecycle metrics |
| sprint-facilitation | `skills/sprint-facilitation/` | Facilitating sprint ceremonies (planning, review, retrospective, standup) |
| ai-lifecycle | `skills/ai-lifecycle/` | AI/ML specific activities at any phase (model dev, experiment, validation, ops) |
| change-control | `skills/change-control/` | Evaluating change requests, classifying significant vs. incremental changes |
| operational-readiness | `skills/operational-readiness/` | Preparing for production deployment, assessing ops readiness before Gate E |
| lifecycle-tailoring | `skills/lifecycle-tailoring/` | Configuring lifecycle for specific product type, adjusting gate depth |
| evidence-management | `skills/evidence-management/` | Collecting, organizing, validating evidence for gates or phase completion |
| retrospective | `skills/retrospective/` | Facilitating or structuring retrospectives (sprint, phase, lifecycle) |
| definition-of-done | `skills/definition-of-done/` | Defining, reviewing, or verifying DoD criteria at story/sprint/phase/release level |
| backlog-management | `skills/backlog-management/` | Shaping, prioritizing, or refining the product backlog; running backlog grooming sessions |
| phase-contract-enforcement | `skills/phase-contract-enforcement/` | Tracking deviations from a signed phase contract, enforcing obligations and escalating blockers |
| responsibility-flows | `skills/responsibility-flows/` | Mapping RACI charts, resolving accountability gaps, producing role responsibility maps |
