# Support

## Getting Help

For usage questions, bug reports, or plugin design discussion:

1. Open a GitHub issue using the provided templates
2. Include reproducible context and expected behavior
3. Reference specific plugin and file paths when possible

## Contact

For private/security-sensitive topics:

- `nuno.salvacao@gmail.com`

## What to Include in Support Requests

- Plugin name and version
- Command or skill being executed
- Exact input used
- Expected vs actual behavior
- Error output (if any)
